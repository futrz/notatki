# notatki
## komendy
* cd - zmiana katalogu
* mkdir - tworzenie katalogu
* ls - wyświetlanie listy plików
* git clone git@github.com:futrz/notatki.git
* git status
* git add .
* git commit -m"opis"
* git push origin master
* git branch
* git checkout - zmiana gałęzi
* git merge nazwa gałęzi
* git log --decorate --all --graph
